# BRthur

## Overview

The aim of this project is to waste a bunch of time writing a componentized
web editor.

Namesake:
![Bea Arthur](http://i.imgur.com/kprOERr.jpg)

## Getting started

Install dependencies:

    npm install
    npm install -g bower
    bower install

Run `grunt karma` to run tests
Run `grunt` to build
